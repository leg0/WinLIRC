TBS-CXT BDA 1.0
============

This plugin is for the CX2388x-based TBS,Prof PC-DVB tuners with BDA-drivers.

In config need select tuner device number (if installed many tuners).

Supported Remotes
=================

NEC-encoded.

Links
=====
